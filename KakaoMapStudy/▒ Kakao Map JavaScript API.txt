■■■ Kakao Map Javascript API ■■■

※ Kakao Map Javascript API 는 키(key) 발급을 받아야 사용할 수 있으며,
   이 키(key)를 발급받기 위해서는 카카오 계정이 필요하다.

○ 키 발급 과정 개요 (b0adb6cf684c4c502462a6673bf822bf)

   1. 카카오 개발자 사이트 접속
   2. 개발자 등록 및 앱 생성
      → https://developers.kakao.com/
   3. 웹 플랫폼 추가
      앱 선택 > 설정 > 일반 > 플랫폼 추가 > 웹 선택 후 추가
   4. 사이트 도메인 등록
      웹 플랫폼 선택, 사이트 도메인 등록
      → 애플리케이션에 우리가 사용할 도메인을 추가해야
         카카오 스크립트를 정상적으로 사용할 수 있으며
         웹 호스팅 및 도메인 서비스를 이용하고 있지 않은 상태에서
         로컬 서버에서 테스트하려면 포트가 포함된 주소를 기재해야 사용할 수 있다.
         (http://localhost:8090)
   5. 『javascript 키』 를 지도 API 의 appkey 로 사용
   6. 애플리케이션 실행

※ 각 Map Api를 운용하는 구조는 똑같으나, 메소드의 이름들이 좀 다르다.
      - 기본적으로 데이터의 처리들을 스크립트를 활용해서 처리한다. (즉, ajax처리가 필요하다.)

   현재는 유료 정책 때문에, 카카오 맵 API를 배울 예정

※ ① 맵 정보를 가지고 있는 서버(카카오)에 우리가 구축한 서버가 요청을 하게 된다.
   ② 일정 요청수가 지나면 정책적으로 유료일텐데... 파이널 프로젝트하면서 그 정도로 많은 요청을 할 것 같지는 않다.
